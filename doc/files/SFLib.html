SF.CheckType (val, typ, level, default)
Checks the type of val. Errors if the types don't match

Parameters:

  • val: The value to be checked.
  • typ: A string type or metatable.
  • level: Level at which to error at. 3 is added to this value. Default is 0.
  • default: A value to return if val is nil.
SF.CreateContext (env, directives, permissions, ops, libs)
Creates a new context. A context is used to define what scripts will have access to.

Parameters:

  • env: The environment metatable to use for the script. Default is SF.DefaultEnvironment
  • directives: Additional Preprocessor directives to use. Default is an empty table
  • permissions: The permissions manager to use. Default is SF.DefaultPermissions
  • ops: Operations quota. Default is specified by the convar "sf_defaultquota"
  • libs: Additional (local) libraries for the script to access. Default is an empty table.
SF.CreateWrapper (metatable, weakwrapper, weaksensitive)
Creates wrap/unwrap functions for sensitive values, by using a lookup table (which is set to have weak keys and values)

Parameters:

  • metatable: The metatable to assign the wrapped value.
  • weakwrapper: Make the wrapper weak inside the internal lookup table. Default: True
  • weaksensitive: Make the sensitive data weak inside the internal lookup table. Default: True

Return values:

  1. The function to wrap sensitive values to a SF-safe table
  2. The function to unwrap the SF-safe table to the sensitive table
