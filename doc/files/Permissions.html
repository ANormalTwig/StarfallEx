Requests a set of permissions. The default implementation of this does nothing, but implementors can use this to change the permissions of a script dynamically

Parameters:

  • permissions: A list of permission names to request

Tables

Permission
Permission format

Fields:

  • name: The name of the permission
  • desc: The description of the permission.
  • level: The abusability of the permission. 0 = low (print to console), 1 = normal (modify entities), 2 = high (run arbitrary lua)
  • value: Boolean. True to allow, false to deny
