Functions

SF.Libraries.AddHook (hookname, func)
Registers a library hook. These hooks are only available to SF libraries, and are called by Libraries.RunHook.

Parameters:

  • hookname: The name of the hook.
  • func: The function to call
SF.Libraries.CallHook (hookname, ...)
Calls a library hook.

Parameters:

  • hookname: The name of the hook.
  • ...: The arguments to the functions that are called.
SF.Libraries.Get (name)
Gets a global library by name

Parameters:

  • name: The name of the library

Return value:

    A metatable proxy of the library
SF.Libraries.Register (name, lib)
Registers a global library. The library will be accessible from any Starfall Instance, regardless of context. This will automatically set __index and __metatable.

Parameters:

  • name: The library name
  • lib: The table containing the library functions.
