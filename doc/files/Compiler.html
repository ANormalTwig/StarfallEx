SF.Compiler.Compile (code, context, mainfile, player, data, dontpreprocess)
Preprocesses and Compiles code and returns an Instance

Parameters:

  • code: Either a string of code, or a {path=source} table
  • context: The context to use in the resulting Instance
  • mainfile: If code is a table, this specifies the first file to parse.
  • player: The "owner" of the instance
  • data: The table to set instance.data to. Default is a new table.
  • dontpreprocess: Set to true to skip preprocessing

Return values:

  1. True if no errors, false if errors occured.
  2. The compiled instance, or the error message.
