SF.Editor.BuildIncludesTable (maincode, codename)
(Client) Builds a table for the compiler to use

Parameters:

  • maincode: The source code for the main chunk
  • codename: The name of the main chunk

Return values:

  1. True if ok, false if a file was missing
  2. A table with mainfile = codename and files = a table of filenames and their contents, or the missing file path.
